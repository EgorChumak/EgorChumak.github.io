Порядок приема топографической и исполнительной съемок на хранение в фонд инженерно-геодезических изысканий Геослужбы по Минскому району

Для сдачи материалов топографических и исполнительных съемок формируется папка, название которой содержит следующие данные : номер заявления-задания, выданный при регистрации, номер объекта, исполнитель работ, населенный пункт, район работ. Пример : «1561 198-18 ИП Иванов Вишневка Дзержинский».
ДЛЯ ТОПОГРАФИЧЕСКИХ СЪЕМОК
Папка должна содержать набор документов:
1. Краткая Пояснительная записка в формате *.doc
2. Картограмма выполненных работ в формате *.dwg
3. Модель в формате *.dwg (для объектов, расположенных на нескольких планшетах)
4. Обновленные планшеты, оформленные в соответствии с существующими требованиями.
ДЛЯ ИСПОЛНИТЕЛЬНЫХ СЪЕМОК
1. Планшеты с нанесенными инженерными коммуникациями (на планшетах в Слой 0 выносится регистрационный номер исполнительной съемки)
2. Исполнительный чертеж, оформленный в соответствии с требованиями существующих ТНПА, в формате *.dwg.
При отсутствии замечаний со стороны специалистов Геослужбы материалы принимаются на хранение, на бумажные копии проставляется штамп Геодезической службы по Минской области.
